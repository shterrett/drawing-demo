# Drawer

This is a demo that parses an **extremely** simple drawing language and draws
lines on the screen.

## Example Script

```
START
Go to 0, 0
Pen Down
Go to 500, 500
Pen Up
END
```

```
START
Go to 250, 0
Pen Down
Go to 500, 250
Go to 250, 500
Go to 0, 250
Go to 250, 0
Pen Up
Go to 350, 350
Pen Down
Go to 150, 150
Pen Up
Go to 150, 350
Pen Down
Go to 350, 150
END
```
